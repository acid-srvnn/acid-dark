### 1. Clone Repo
* ` git clone https://github.com/acid-srvnn/acid-dark.git `

### 2. Open with VS Code

### 3. Install Node dependencies
* ` npm install `

### 4. Start auto build
* `Ctrl+Shift+B` on VSCode to start auto build

### 5. Run local server
* Navigate to build folder and `npm run serve`
